# cargo-help(1)
{==+==}

{==+==}


{==+==}
## NAME
{==+==}
## 定义
{==+==}


{==+==}
cargo-help - Get help for a Cargo command
{==+==}
cargo-help - 对于Cargo命令获得帮助。
{==+==}


{==+==}
## SYNOPSIS
{==+==}
## 概要
{==+==}


{==+==}
`cargo help` [_subcommand_]
{==+==}

{==+==}


{==+==}
## DESCRIPTION
{==+==}
## 说明
{==+==}


{==+==}
Prints a help message for the given command.
{==+==}
打印指定命令的帮助信息。
{==+==}


{==+==}
## EXAMPLES
{==+==}
## 示例
{==+==}


{==+==}
1. Get help for a command:

       cargo help build
{==+==}
1. 获取命令帮助:

       cargo help build
{==+==}


{==+==}
2. Help is also available with the `--help` flag:

       cargo build --help
{==+==}
2. 也可以通过 `--help` 参数获得帮助:

       cargo build --help
{==+==}
